# ADR-0009: No Gamification in MVP

## Status

Proposed

## Context

The core problem is skill maintenance through deliberate refactoring practice, not engagement mechanics.

## Decision

Exclude gamification features such as badges, points, and leaderboards from MVP.

## Consequences

- Positive: keeps the MVP focused on the learning loop
- Positive: reduces product and UI scope
- Negative: short-term engagement features are deferred
- Negative: differentiation must come from exercise quality rather than reward systems
